# co2-emissions

## Requirements
* node
* npm
* PostgreSQL

## Installation
### dependencies
```bash
npm install
```

### database migrations
```bash
npm run migrate
```

## Running

### development
```bash
npm run start:dev   # frontend file watch
npm run server:dev  # development server
```

### production
```bash
npm start           # compile bundle and start server
```
